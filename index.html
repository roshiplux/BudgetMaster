This is a demo version. To enable Google Drive sync, you need to set up your own Google Cloud project.

1 Create Google Cloud Project

  1. Go to Google Cloud Console
  2. Click "New Project" and give it a name (e.g., "BudgetMaster")
  3. Select your new project from the dropdown

2 Enable Google Drive API

  1. In the Cloud Console, go to "APIs & Services" ‚Üí "Library"
  2. Search for "Google Drive API"
  3. Click on it and press "Enable"

3 Create Credentials

  1. Go to "APIs & Services" ‚Üí "Credentials"
  2. Click "Create Credentials" ‚Üí "OAuth 2.0 Client ID"
  3. Choose "Web application" as the application type
  4. Add your domain to "Authorized JavaScript origins"
  5. Copy the Client ID that's generated

4 Create API Key

  1. In "Credentials", click "Create Credentials" ‚Üí "API Key"
  2. Copy the API Key that's generated
  3. Optionally, restrict the key to Google Drive API only

5 Update the Code

Replace these lines in the JavaScript code:

// Replace these with your actual credentials:
this.CLIENT_ID = 'YOUR_CLIENT_ID_HERE';
this.API_KEY = 'YOUR_API_KEY_HERE';

‚ùì Frequently Asked Questions

Is my data secure?

Yes! Your data is stored locally in your browser and optionally synced to your personal Google Drive. We never have access to your financial information.

Can I use this offline?

Yes! The app works completely offline. Google Drive sync is optional and only used when you're online.

What happens if I clear my browser data?

If you have Google Drive sync enabled, you can restore your data. Otherwise, make sure to export your data regularly as backup.

Can I connect my bank account directly?

No, this is a manual tracking app for security reasons. You need to enter transactions manually, which gives you full control over your data.
